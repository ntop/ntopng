Interfaces
##########

ntopng uses interfaces to perform its tasks. One or more interfaces are required by ntopng to properly execute. Interfaces are specified to ntopng using option :code:`-i`. Multiple types of interface exist.

This section introduces the concept of interfaces and discusses differences, pros and cons of each interface type.
